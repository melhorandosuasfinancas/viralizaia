# ViralizaIA

Transforme vídeos do YouTube em cortes virais para TikTok, Instagram e Facebook — automaticamente com IA.

## Estrutura

```
ViralizaIA/
├── backend/    → Node.js + Express + FFmpeg + Whisper
└── frontend/   → Next.js 14 + Tailwind CSS
```

## Deploy

**Backend → Railway** (usa Dockerfile automático)  
**Frontend → Vercel** (conecta o repositório)

## Variáveis de ambiente

Ver `backend/.env.example` e `frontend/.env.local.example`
